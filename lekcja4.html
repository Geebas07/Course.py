Lekcja 4: Typy zmiennych oraz wartości przeznie przyjmowane cz.2 + if

Cele lekcji:

  1. Nauka reszty zmiennych
  2. Nauka obsługi funkcji if

Kolejną ważną zmienną często wykorzystywaną przez programistów jest bool (boolean). Zmienna ta przyjmuje tylko dwie wartości false(0) oraz true(1). Czyli po przetłumaczeniu na polski prawda lub fałsz. Na razie bool za bardzo Ci się nie przyda, więc pomińmy go. Przejdźmy natomiast do funkcji if. Funkcja ta sprawdza czy podany warunek jest prawdziwy. Jeżeli warunek jest prawdziwy, to funkcja aktywuje kod do niej przypisany. Przeanalizuj poniższy przykład.

Przykład:

liczba = int(input("Podaj jakąś liczbę: "))
if liczba>5:
print("Liczba jest większa od 5.")

Wynik:

Podaj jakąś liczbę: 6
Liczba jest większa od 5.

Funkcja if może również aktywować dany kod, jeżeli warunek nie zostanie spełniony. Służy do tego else(ang. - ponieważ). Else używa się odrazu po napisaniu funkcji if oraz okresleniu jej warunku oraz napisaniu kodu do aktywacji dla prawdziwego warunku. Przeanalizuj poniższy przykład wykorzystania tejże funkcji.

Przykład:

liczba = int(input("Podaj jakąś liczbę: "))
if liczba>10:
print("Liczba jest większa od 10.")
else:
print("Liczba jest mniejsza od 10.")

Wynik:

Podaj jakąś liczbę: 8
Liczba jest mniejsza od 10.
