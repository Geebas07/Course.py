Lekcja 9: Pętla while

Cele lekcji:

  1. Poznanie zasady działania pętli while
  2. Przykłady zastosowań pętli while

Pętla while w języku Python służy do powtarzania bloku kodu tak długo, jak długo warunek określony przed pętlą jest spełniony. Wykonuje się następująco: najpierw sprawdza się warunek.

Przykład:

count = 0

while (count < 3):

count = count + 1

print("Hello World")

Wynik:

Hello World
Hello World
Hello World

Funkcja while bywa przedatna w momencie gdy chcemy aby użytkownik wprowadził określoną wartość.

Przykład:

liczba = int(input("podaj liczbę większą lub równą 5: ))

while liczba < 5:

liczba = int(input("podaj liczbę większą lub równą 5: ))

else:

print("poprawna liczba)

Wynik:

podaj liczbę większą lub równą 5: 2
podaj liczbę większą lub równą 5: 7
poprawna liczba
