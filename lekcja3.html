Lekcja 3: Typy zmiennych oraz wartości przeznie przyjmowane cz.1

Cele lekcji:

  1. Nauka zmiennych liczbowych
  2. Nauka zmiennych przyjmujących znaki

Poznałeś juz dwie zmienne: int oraz string. Już tłumaczę czym jest string. Prawdopodobnie jeszcze nie wiesz czym jest string więc wytłumaczę. String to zmienna, która przyjmuje znaki oraz ciągi znaków niezależnie od wielkości liter. Kolejnymi zmiennymi, których jeszcze nie poznałeś są float oraz double. Zmienne te w przeciwieństwie do inta przyjmują także liczby niecałkowite takie jak np.: 0., 0.64. 3.14. Różnica między float a double jest jedna; double może mieć maksymalnie aż dwa razy więcej liczb po przecinku niż float. A teraz zróbmy przykład z wykorzystaniem zmiennej float.

Przykład:

niecalkowita = float(input("Podaj jakąkolwiek liczbę niecałkowitą: "))
print("Podana przez ciebię liczba niecałkowita: "+ niecalkowita)

Wynik:

Podaj jakąkolwiek liczbę niecałkowitą: 3.1459
Podana przez ciebię liczba niecałkowita: 3.1459

Kolejną ważną zmienną jest "char", zmienna ta przyjmuje tylko jeden znak oraz nie przyjmuje liczb. Stwórzmy teraz przykładowy kod proszący o pierwszą literę z imienia użytkownika.

Przykład:

litera = char(input("Podaj pierwszą literę swojego imienia: "))
print("Pierwsza litera twojego imienia to: " + litera)

Wynik:

Podaj pierwszą literę swojego imienia: A
Pierwsza litera twojego imienia to: A
